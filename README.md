# 项目启动步骤

#### 第一步：克隆项目

```
git clone https://github.com/JefferyXZF/shopAdmin.git
```

#### 第二步：安装依赖

```
进入shopAdmin目录下：
执行命令：cnpm install 安装依赖（使用npm install可能安装失败）
```

#### 第三步：启动项目

```
npm run dev
```
